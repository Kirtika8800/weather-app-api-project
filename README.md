# 🌦 Weather Analytics Dashboard

## 📌 Overview

This project is an interactive **Weather Analytics Dashboard** built using Power BI. It integrates weather data (via API) and presents key insights such as temperature, humidity, and wind speed across multiple cities through intuitive visualizations.

The dashboard is designed to provide a clear understanding of weather patterns and comparisons between different locations.

---

## 🎯 Objectives

* Analyze weather conditions across different cities
* Compare temperature, humidity, and wind speed
* Identify trends and patterns in weather data
* Present insights using interactive visualizations

---

## 🗂 Dataset / Data Source

* Weather data collected using API (e.g., OpenWeatherMap / WeatherAPI)
* Includes:

  * City Name
  * Temperature (°C)
  * Humidity (%)
  * Wind Speed
  * Date & Time

---

## 🛠 Tools & Technologies Used

* Power BI (Dashboard Development)
* API Integration (Data Extraction)
* DAX (Basic Calculations)
* Data Cleaning & Transformation

---

## 📊 Dashboard Features

### 🔹 KPI Cards

* Average Temperature
* Humidity
* Wind Speed

### 🔹 Visualizations

* 📊 Bar Chart → City-wise temperature comparison
* 📈 Area Chart → Humidity trends
* 🗺 Map → Geographical temperature distribution
* 📉 Additional comparison charts

### 🔹 Interactivity

* Filters for city selection
* Dynamic visuals based on user interaction

---

## 🧠 Key Insights

* Hyderabad shows the highest temperature among selected cities
* Temperature differences across cities are clearly visible
* Humidity levels remain relatively stable
* Visual patterns help identify weather variations quickly
